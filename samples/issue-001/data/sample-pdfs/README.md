# Sample PDFs for Testing

This folder should contain 2-3 sample PDF documents for demo purposes.

## Suggested Sample Documents

Since we can't redistribute copyrighted PDFs, create your own test documents:

### Option A: Generate with Copilot Chat
Paste this into Copilot Chat or any AI assistant:

```
Create a 3-page regulatory compliance report for a fictional company "Contoso Corp"
covering Q3 2025. Include sections on: data privacy compliance, financial controls audit,
and environmental sustainability metrics. Include specific numbers and findings.
Export as PDF.
```

Repeat with variations:
- A vendor contract with payment terms and SLA requirements
- A technical architecture document with system requirements

### Option B: Use Public Domain PDFs
- SEC EDGAR filings (public company 10-K reports): https://www.sec.gov/cgi-bin/browse-edgar
- Government reports from data.gov
- Academic papers from arXiv

### What You Need
- At least 2 PDFs with 5+ pages each (to demonstrate chunking)
- Documents that share some common topics (to demonstrate cross-document querying)
- Content you're comfortable using in a demo setting

### Testing Queries
After uploading your PDFs, try these queries:
1. "Summarize the key findings in [document name]"
2. "What compliance requirements are mentioned?"
3. "Compare the financial figures across the uploaded documents"
4. "What are the main risks identified?"
