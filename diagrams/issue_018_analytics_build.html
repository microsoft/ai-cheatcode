Golden Test Runner
Weekly scheduled flow: sends 20 test questions to agent via HTTP, compares responses to expected answers using Azure OpenAI similarity scoring
🚨
Regression Alert Flow
If similarity drops below threshold → Teams notification: "3 of 20 golden test answers degraded this week — review recent knowledge changes"

Telemetry Logger

Teams Notification
Regression alerts, weekly digest of negative feedback, and weekly test summary delivered to agent owner's Teams channel
🔑 Architectural Insight
Add analytics to any existing Copilot Studio agent in one session. Enable Application Insights (Settings → Advanced), add a thumbs-up/down Adaptive Card, build 3 KQL queries (sessions/day, avg confidence, feedback ratio), and pin to a Power BI dashboard. The golden test runner catches regressions before users do.
