🤖 AI Builder Classifier
🧠
Custom Classification Model
Trained on 50–100 examples per category. Returns classification + confidence score per query.
📊
Training Data
Real user queries labeled by category:
  • ✅ General — "What's the PTO policy?"
  • ⚠️ Sensitive — "What's the salary range for L6?"
  • 🚫 Restricted — "What's John's salary?"
📈
Confidence Score
Each classification returns 0–100% confidence. Below 70% → log for human review and default to sensitive handling.

🗃️ Dataverse Feedback & Retraining
📊
Classification Logs
Every query logged: input text, classification result, confidence score, user thumbs up/down feedback. Enables model performance tracking.
🔄
Retraining Pipeline
Monthly Power Automate flow exports low-confidence classifications (<70%) as retraining candidates. Model improves with each cycle.
📈
Power BI Dashboard
Guardrail trigger rates, false positive rates, escalation volume, average confidence scores. Track guardrail health over time.
🔑 Build Insight
The classifier topic runs BEFORE generative answers — every query is classified before the agent processes it. This is an architectural guardrail, not a prompt-based one. Prompt injection can't bypass a topic that never reaches the LLM. Cost optimization: only validate sensitive classifications; skip validation for general to halve Azure OpenAI costs.
