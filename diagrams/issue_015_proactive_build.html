Building Proactive Agents with Copilot Studio
Issue #015 — Pattern: Scheduled Scan → AI Classification → Adaptive Card → Feedback Loop
▶ Proactive Scan & Delivery Pipeline
⚡ Power Automate Scheduled Scan
🕐
Recurrence Trigger
Daily at 8 AM user's local time. Automated scan with no user interaction required.
🔍
Data Source Queries
Parallel queries to connected systems:
  • 📋 SharePoint — overdue tasks, stale documents
  • 🗄️ Dataverse — stale records, pending items
  • 📅 Planner — upcoming deadlines, blocked tasks
🤖
Azure OpenAI Summarization
Collects all findings into a JSON array, sends to Azure OpenAI for natural-language summary and severity classification.
  • 🔴 Critical — "Compliance cert expires tomorrow"
  • 🟡 Warning — "3 items overdue by 2+ days"
  • 🟢 Info — "New document uploaded to Project X"
Classified alerts
💬 Microsoft Teams Delivery
📨
Adaptive Card Delivery
Critical → sent immediately. Warnings → batched into daily digest card at 8 AM.
🔴 CRITICAL Compliance Certificate Expiring
Your SOC 2 compliance certificate for Project Falcon expires tomorrow (April 15). Renewal requires manager approval and takes 48 hours.
📊
Daily Digest
Single card summarizing all warnings: grouped by project, counts, and quick links. Sent at 8 AM.
"Ask the Agent" deep link
🤖 Copilot Studio Agent
🔔
"Show My Alerts" Plugin
Plugin action queries the Dataverse alert log. When user opens agent:
  • 📋 "You have 2 unacknowledged alerts from yesterday"
  • 🗂️ Lists alerts by severity with one-line summaries
  • 💬 "Want me to walk through them?"
💬
Conversational Follow-Up
User can ask questions about any alert: "Tell me more about the compliance cert" — agent pulls context from Dataverse + source systems.

🔄 Feedback Loop Tuning
⏸️
Snooze Action
User clicks "Snooze 24h" on Adaptive Card → Power Automate writes snooze_until to Dataverse → alert resurfaces tomorrow.
🚫
Dismiss Action
User dismisses alert → status set to dismissed. Dismissed patterns feed classification tuning.
📉
Frequency Tuning
"User dismissed 5 'info' alerts this week" → reduce info alert frequency for this user. System learns what matters.
Tuning signal to classifier
⚡ Power Automate Scan Config
⚙️
Classification Prompt
Azure OpenAI prompt updated with user feedback patterns: suppress frequently-dismissed categories, boost categories users engage with.
📅
Schedule Adjustment
Snooze patterns may adjust scan frequency: if all items are snoozed, reduce to weekly. If critical items found daily, keep daily cadence.
💡 Always-On Value
The agent delivers value even when no one is in the chat window. Users get a personalized project health check every morning — without anyone having to remember to ask for it.
🔑 Build Sequence
Day 1: Scheduled scan + one data source + Teams card. Day 2: AI classification + severity routing. Day 3: Snooze/dismiss feedback + "Show My Alerts" plugin. Ship incrementally — each stage adds value.
