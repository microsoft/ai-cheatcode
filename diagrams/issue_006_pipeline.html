🔑 Architectural Insight
These are two separate, independently deployable agents connected through an approval gate and shared SharePoint storage. The Scribe Agent can run without the Library Agent (just summaries). The Library Agent can index content from any source — not just meetings. Build this as a 2.5-day hackathon with the customer.
