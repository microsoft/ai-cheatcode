Layer Classification
Each link classified by abstraction distance:
1
Trails
Direct local references — "Function X calls Y"
2
Roads
Thematic connections — "Rate limiting ↔ Token budgets"
3
Sea Routes
Cross-domain abstractions — "Auth in Proj A ↔ Compliance in Proj B"
4
Air Routes
Conceptual parallels — "Design patterns ↔ Regulatory frameworks"
📋
Context Redundancy
Each node stamped with: origin project, author, date, key decisions, related artifacts, summary context. Every node can reconstruct the broader picture alone.

🔮 Holographic Property
Every node carries redundant context. Remove any node and the remaining nodes still reconstruct the answer — at lower fidelity, but without blind spots. Storage trades for resilience.
🗺️ Layered Traversal
Code questions stay on Trails (fast, precise). Strategy questions take Air Routes (broad, conceptual). The agent chooses depth based on query intent — not a fixed search radius.
⚡ Graceful Degradation
SharePoint offline? Repo archived? Chat history deleted? Remaining nodes still have the context stamps. Fidelity degrades; coherence doesn't. The opposite of flat keyword search.
