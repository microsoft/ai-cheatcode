Depth & Elevation
System

A 6-level luminance stacking model with enhanced glass surfaces, cascade shadows, and dimensional diagram interactions. Hover and click everything — this prototype is interactive.

01 / 04
Luminance Stacking Model
Six elevation levels form a layered lighting system. Lower levels are darker — elements rise toward the viewer by increasing luminance and shadow depth. Hover each row to see the layer react.
Hover rows to activate
L0 · #0E0E1F
Void
The abyss — the absolute floor. Nothing sits below this level.
Use: page background, outer canvas
L1 · #1B1B3A
Surface
Nav bars, side panels, and ambient containers. The ambient foundation above the void.
Use: global-nav, sidebar, hero sections
L2 · rgba white 4%
Glass (Rest)
Cards and panels at rest. Frosted glass with 20px blur — visible but quiet.
Use: pattern cards, diagram zones, panel sections
L3 · rgba white 7%
Glass (Hover)
Cards responding to cursor proximity. Slightly brighter glass surface + border brightens.
Use: card :hover state, focused diagram elements
L4 · Purple Glow
Glow
Highlighted / active elements. Purple shadow halo pulls attention without harsh borders.
Use: active step, selected card, CTA buttons
L5 · Float
Float
Modals, lifted cards, tooltip surfaces. Strong shadow cascade + bright border = visually closest to viewer.
Use: context panels, modals, tooltips, active overlays
✦
The stacking rule: Each level increases luminance and shadow weight, never color saturation alone. Elevation is felt through light, not hue — keeping the palette tight while creating clear spatial hierarchy.
Visual depth demo — 6 layers stacked in 3D space (hover to inspect)
L0 Void — #0E0E1F — page canvas
L1 Surface — #1B1B3A — nav & panels
L2 Glass Rest — rgba(255,255,255, 4%) + blur
L3 Glass Hover — rgba(255,255,255, 7%) + blur
L4 Glow — purple halo shadow
L5 Float — strong cascade + bright border
